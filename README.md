# Unitpay PHP SDK

> Language: Русский | [English](README_en.md)

Данное SDK позволяет работать с платёжной системой Unitpay на PHP.

## Требования

- PHP 7.1 или новее;
- Composer.

## Установка

Просто выполните следующую команду:

```
composer require zkelo/unitpay-php-sdk
```
