# Unitpay PHP SDK

Данное SDK позволяет работать с платёжной системой Unitpay на языке программирования PHP.

## Требования

- PHP 7.1 или новее;
- Composer.

## Установка

Для установки необходимо ввести следующую команду:

```
composer require zkelo/unitpay-php-sdk
```
