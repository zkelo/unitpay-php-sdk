# Unitpay PHP SDK

> Language: [Русский](README.md) | English

This SDK allows you to work with Unitpay payment system on PHP.

## Requirements

- PHP 7.1 or newer;
- Composer.

## Install

Just execute this command:

```
composer require zkelo/unitpay-php-sdk
```
